# 📚 StudyFlow – Smart Study Planner

StudyFlow is a modern web-based study planning application that helps students organize subjects, track study progress, generate study schedules, and monitor learning analytics.

## ✨ Features

### 📖 Subject Management

* Add subjects with exam dates and required study hours
* Track hours studied for each subject
* Delete subjects when no longer needed
* Automatic countdown to exam dates

### 📅 Smart Study Schedule

* Generate a personalized study schedule
* Prioritize subjects based on exam deadlines
* Built-in break recommendations
* Easy schedule reset option

### 📊 Study Analytics

* Overall study progress tracking
* Total subjects overview
* Urgent exam detection
* Total study hours completed
* Individual subject progress visualization

### 💾 Data Persistence

* Uses Local Storage to save data
* Study information remains available after refreshing the browser

### 🎨 Modern UI

* Glassmorphism design
* Animated gradient background
* Responsive layout
* Interactive progress indicators
* Smooth notifications and transitions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API
* Font Awesome Icons
* Google Fonts (Poppins)
